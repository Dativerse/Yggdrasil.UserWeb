# Yggdrasil.UserWeb

Part of the Yggdrasil.NET Distributed IOT System.

## Development

This is a submodule of the main Yggdrasil.NET repository.

For development instructions, see the [main repository](https://github.com/dativerse/Yggdrasil.NET).

## Getting Started

1. This repository is meant to be used as a submodule
2. Clone the main repository with: `git clone --recursive git@github.com:dativerse/Yggdrasil.NET.git`
3. See DEVELOPMENT.md in the main repository for setup instructions
